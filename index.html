About the Rhythnic slideshow plugin

The Rhythnic slideshow jquery plugin is a lightweight, customizable jquery slider plugin made for image slideshows. Users of touch screens can navigate the slideshow by swiping left or right. Desktop users will see the navigation icons appear when their mouse hovers over the slideshow. The plugin is dependent upon jQuery, Modernizr, Font Awesome, and the touchSwipe plugin for jQuery. I'll explain how to setup the plugin and incorporate all of these technologies in the instructions below. Setting up the Rhythnic slideshow plugin does require a little knowledge of HTML and CSS.

How to Use the Image Slideshow Plugin

Putting the Rhythnic Image Slideshow on your page is a combination of HTML, CSS, and a little bit of javascript. Below I've broken it down into those sections so that you know exactly what to do to put the slideshow on your page. I'll also show you how to incorporate the technologies that the slideshow depends on, such as jQuery, Font Awesome, Modernizr, and touchSwipe.

The HTML

Let's go over the code for the HTML file that will include the slideshow. We'll start at the top and work down.

The HEAD
Link to font awesome to enable the icons to display. Put slideshow.css in your css folder and link to it. The file main.css is whatever your primary css file is for your page. If you copy and paste the rules from slideshow.css into your main.css, you can omit the link to slideshow.css. Include the Modernizr script in the head. Modernizr will detect touch screens, and the slideshow will look and act accordingly.

The HEAD: CSS, Modernizr, Font Awesome, Viewport
  1. <head>
  2.   <meta charset="UTF-8">
  3.   <meta name="viewport" content="width=device-width, initial-scale=1" />
  4.   <title>Your page title</title>
  5.   <link href="//netdna.bootstrapcdn.com/font-awesome/4.0.3/css/font-awesome.css" rel="stylesheet">
  6.   <link href="/css/slideshow.css" rel="stylesheet">
  7.   <link href="/css/main.css" rel="stylesheet">
  8.   <script src="/js/modernizr.custom.78546.js"></script>
  9. </head>

Let's look at our list of images next. The slideshow works by manipulating a list on your site. Include the list inside of a div with a class of slideshow and place that div on your page whereever you want the slideshow to be. Rhytnic's slideshow is made for use with images, but you could tweak it for other purposes. Any number of images is fine, but they should be the same size.

A list of images inside of a div with a class of slideshow
  1. <div class="slideshow">
  2.   <ul>
  3.     <li><img src="/img/image1.jpg" alt="Description of image one"></li>
  4.     <li><img src="/img/image2.jpg" alt="Description of image two"></li>
  5.     <li><img src="/img/image3.jpg" alt="Description of image three"></li>
  6.     <li><img src="/img/image4.jpg" alt="Description of image four"></li>
  7.   </ul>
  8. </div>

If you want captions, just include a caption inside of a paragraph tag inside of the list element.

An image with a caption
  1. <li>
  2.   <img src="/img/image1.jpg" alt="Description of image one">
  3.   <p>Caption for image one.</p>
  4. </li>

Copy this div for the slideshow navigation and paste it below the unordered list of images, but still inside of the div with a class of slideshow.

Slideshow navigation div
  1. <div class="slideshow-nav">
  2.   <i data-dir="prev" class="skip fa fa-chevron-left"></i>
  3.   <i class="play fa fa-play"></i>
  4.   <i data-dir="next" class="skip fa fa-chevron-right"></i>
  5. </div>

The Rhythnic Image Slideshow uses Font Awesome, a popular brand of font icons, which work by assigning classes to i elements. Any i element in the slideshow with an "fa" class is a Font Awesome icon. The other class names and the "data-dir" attribute assist in the functioning of the plugin. I chose Font Awesome because it's free and very user friendly.

I chose to include the entire Font Awesome collection by linking to it in the head section for a few reasons, even though we're downloading more than is necessary. The first reason is that it keeps the slideshow package simple. Also, the less I integrate the font icon system, the easier it is for developers to integrate the slideshow plugin with the other font icons on their site, whether they are using Font Awesome or another font icon system. You're welcome to use another font icon system, but you have to change the class names to match whatever you decide to use. You'll also need to specify the new class names in the options object when you create the jQuery object. This is explained better in the options section below. Beginning developers will also benifit from the simplicity of linking to Font Awesome in the head.

Under the navigation, include a message that tells touch screen users to swipe.

Swipe message
  1. <p class="swipe-msg"><i class="fa fa-long-arrow-left"></i> swipe <i class="fa fa-long-arrow-right"></i></p>

The CSS

We covered how to include the css in the HEAD section of your page. Next we'll look at styling or positioning the slideshow.

The slideshow.css file won't need to be modified, but you can if you want to. Each image will automatically fit the width of the container. For the best presentation, please make all of the images files the same width and height. If your using images with different sizes, consider specifying the height of the .slideshow container so that it stays consistent.

Include a css selector in your main css file for the slideshow. Use it to manpulate the size, position, and style of the slideshow container. The navigation and captions are sized in em, so you can increase or decrease their size with font-size in the .slideshow selector.

.slideshow selector in your main css file
  1. .slideshow {
  2.   width: 600px; /* width of container */
  3.   font-size: 100%;  /* adjust to control the sizing of the icons and text in slideshow*/
  4. }

In the absence of javascript, the slideshow is a horizontal scrolling list of images. In this instance, the images don't take the width of the slideshow container. They are just their natural size. If you'd like to alter their size in the event that there is no javascript, please add a css rule.

Force image size
  1. .slideshow img {
  2.   width: 600px; /* ideal display width of each image */
  3. }

The Javascript

Put slideshow.min.js, modernizr.custom.80160.js, and jquery.touchSwipe.min.js in your script directory (js). The Rhythnic slideshow depends on jQuery and the other js files, so you have to link to them before linking to the slideshow.js file. The best place to include scripts, Modernizr excluded, is at the bottom of the body container.

The first 2 lines get jQuery from Google's CDN with jquery.com as a fallback. In the 3rd line, Modernizr tests for touch screens, and if the page was loaded on a touch screen, the touchSwipe plugin is imported. The 4th line imports the Rhythnic slideshow plugin.

Link to jQuery and slideshow.js
  1. <script src = "//ajax.googleapis.com/ajax/libs/jquery/1.10.2/jquery.min.js"></script>
  2. <script>window.jQuery || document.write('<script src="//codeorigin.jquery.com/jquery-1.10.2.min.js"><\/script>')</script>
  3. <script>Modernizr.touch && document.write('<script src="js/jquery.touchSwipe.min.js"><\/script>')</script>
  4. <script src="/js/slideshow.min.js"></script>

On a sidenote, for anyone who knows about Modernizr.load, I chose "Modernizr.touch &&" instead because Modernizr.load doesn't work with iOS devices 4.3 and earlier.

This code will create the slideshow for all divs that have a class of slideshow The code is placed inside of $window.load() to ensure that all the images have downloaded before the slideshow is created.

Create the jQuery object for slideshow
  1. <script>
  2. $(window).load(function() {
  3.   $('div.slideshow').slideshow();
  4. });
  5. </script>

Options

Currently, there are 2 options to control playback. You can have the slideshow auto-start when the page loads and you can specify the time between each picture when the slideshow is on play. In regard to the options object, you can specify any options in any order, including or omitting any at will, as long as you use the correct key names, such as play and timer.

The values specified in this code are the default values. You can replace false with true to enable autoplay, and you can replace 5000 with however many milliseconds you would like to be in between each picture.

Create the jQuery object for slideshow with options set
  1. <script>
  2. $(window).load(function() {
  3.   $('div.slideshow').slideshow( { play: false, timer: 5000 });
  4. });
  5. </script>

If you have decided to use another font icon system, please specify the new play and pause class names in the options object.

Specifying class names for play and pause icons
  1. <script>
  2. $(window).load(function() {
  3.   $('div.slideshow').slideshow( { playClass: "font-icon-play", pauseClass: "font-icon-pause" });
  4. });
  5. </script>

Rhythnic Image Slideshow is Open Source

The origins of this humble little slideshow are from the Tuts Premium course, 30 Days to Learn jQuery. The course is free at the time I'm writing this, and it really taught me a lot about jQuery and about how to organize javascript code using dictionaries. I highly recommend it.

This plugin is open source and there is a public repository for it on Bitbucket called rhythnic/image-slideshow. I will continue to develop it here and there in the future as time allows, but anyone is able to fork the git project and continue development. There are many image slideshow plugins around, but I chose to create one more for the purpose of understanding the process of making a plugin.

The Rhythnic Image Slideshow jQuery Plugin is released under the open source software license. Anyone may download it for free, change it at will, and use it for commercial purposes.
