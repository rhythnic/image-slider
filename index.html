How to Use the Image Slideshow Plugin

Putting the Rhythnic Image Slideshow on your page is a combination of HTML, CSS, and a little bit of javascript. Below I've broken it down into those sections so that you know exactly what to do to put the slideshow on your page.

The Zip File

The zip file contains the following files:

Index.html
If you open this in your browser, you'll see the same page that you're viewing right now, or at least something very similar.
slideshow.min.js
This is the minimized javascript file for the image slider. Link to it in your page after linking to jQuery.
slideshow.js
You won't use this file, most likely, but it's here in case you want to see the javascript code for the slider or alter the slider.
slideshow.css
This is the CSS for the image slider. Link to it in the HTML head section or copy and paste the code into your CSS file.

The HTML

The slider works by manipulating a list of images on your site, so a good place to start is to make the list of images. Include the list inside of a div with a class of slideshow and place that div on your page whereever you want the slideshow to be. Any number of photos is fine.

A list of images inside of a div with a class of slideshow
  1. <div class="slideshow">
  2.   <ul>
  3.     <li><img src="/img/image1.jpg" alt="Description of image one"></li>
  4.     <li><img src="/img/image2.jpg" alt="Description of image two"></li>
  5.     <li><img src="/img/image3.jpg" alt="Description of image three"></li>
  6.     <li><img src="/img/image4.jpg" alt="Description of image four"></li>
  7.   </ul>
  8. </div>

If you want captions, just include a caption inside of a paragraph tag inside of the list element.

An image with a caption
  1. <li>
  2.   <img src="/img/image1.jpg" alt="Description of image one">
  3.   <p>Caption for image one.</p>
  4. </li>

Copy this div for the slideshow navigation and paste it below the unordered list of images, but still inside of the div with a class of slideshow.

Slideshow navigation div
  1. <div class="slideshow-nav">
  2.   <i data-dir="prev" class="skip fa fa-chevron-left"></i>
  3.   <i class="play fa fa-play"></i>
  4.   <i data-dir="next" class="skip fa fa-chevron-right"></i>
  5. </div>

The Rhythnic Image Slideshow uses Font Awesome, a popular brand of font icons. Any class in the slideshow navigation div that starts with "fa" is a Font Awesome class, which results in the icon being displayed. The other class names and the "data-dir" assist in the functioning of the plugin.

In order for the font icons to appear, you need to include a link to the Font Awesome css in the head section of your HTML.

Link to Font Awesome in the head section
  1. <link href="//netdna.bootstrapcdn.com/font-awesome/4.0.3/css/font-awesome.css" rel="stylesheet">

You're welcome to use another font icon system, but you have to change the class names in the slideshow-nav div to match whatever you decide to use. You'll also need to specify the new class names in the options object when you create the jQuery object. This is explained better in the options section below.

The final, complete slideshow div should look something like this.

The complete slideshow div
  1. <div class="slideshow">
  2.   <ul>
  3.     <li><img src="/img/image1.jpg" alt="Description of image one"></li>
  4.     <li><img src="/img/image2.jpg" alt="Description of image two"></li>
  5.     <li><img src="/img/image3.jpg" alt="Description of image three"></li>
  6.     <li><img src="/img/image4.jpg" alt="Description of image four"></li>
  7.   </ul>
  8.   <div class="slideshow-nav">
  9.     <i data-dir="prev" class="skip fa fa-chevron-left"></i>
  10.     <i class="play fa fa-play"></i>
  11.     <i data-dir="next" class="skip fa fa-chevron-right"></i>
  12.   </div>
  13. </div>

The CSS

Either copy and paste all of the css into your own css code, or put the slideshow.css file in your css directory and link to the slideshow.css file in the head.

Link to slideshow.css
  1. <link href="/css/slideshow.css" rel="stylesheet">

The slideshow.css file won't need to be modified, but you can if you want to. Each image will automatically fit the width of the container. For the best presentation, please make all of the images files the same width and height. If your using images with different sizes, consider specifying the height of the .slideshow container so that it stays consistent.

Include a css selector in your main css file for the slideshow. Use it to manpulate the size and positioning of the slideshow container. The navigation and captions are sized in em, so you can increase or decrease their size with font-size in the .slideshow selector.

.slideshow selector in your main css file
  1. .slideshow {
  2.   width: 600px; /* width of container */
  3.   font-size: 100%;  /* adjust to control the sizing of the icons and text in slideshow*/
  4. }

In the absence of javascript, the slideshow is a horizontal scrolling list of images. In this instance, the images don't take the width of the slideshow container. They are just their natural size. If you'd like to alter their size in the event that there is no javascript, please add a css rule.

Force image size
  1. .slideshow img {
  2.   width: 600px; /* ideal display width of each image */
  3. }

The Javascript

Put slideshow.js in your script directory (js). The Rhythnic slideshow depends on jQuery, so you have to link to jQuery before linking to the slideshow.js file. The best place to include scripts and links to scripts is at the bottom of the body container.

This code will attempt to download jQuery from Google's CDN, and if that fails it will attempt to get it from jQuery.

Link to jQuery and slideshow.js
  1. <script src = "//ajax.googleapis.com/ajax/libs/jquery/1.10.2/jquery.min.js"></script>
  2. <script>window.jQuery || document.write('<script src="//codeorigin.jquery.com/jquery-1.10.2.min.js"><\/script>')</script>
  3. <script src="/js/slideshow.min.js"></script>

This code will create the slideshow for all divs that have a class of slideshow. Be sure to put this after your lines to download jQuery and slideshow.min.js. The code is placed inside of $window.load() to ensure that all the images have downloaded before the slideshow is created.

Create the jQuery object for slideshow
  1. <script>
  2. $(window).load(function() {
  3.   $('div.slideshow').slideshow();
  4. });
  5. </script>

Options

Currently, there are 2 options to control playback. You can have the slideshow auto-start when the page loads and you can specify the time between each picture when the slideshow is on play. In regard to the options object, you can specify any options in any order, including or omitting any at will, as long as you use the correct key names, such as play and timer.

The values specified in this code are the default values. You can replace false with true to enable autoplay, and you can replace 5000 with however many milliseconds you would like to be in between each picture.

Create the jQuery object for slideshow with options set
  1. <script>
  2. $(window).load(function() {
  3.   $('div.slideshow').slideshow( { play: false, timer: 5000 });
  4. });
  5. </script>

If you have decided to use another font icon system, please specify the new play and pause class names in the options object.

Specifying class names for play and pause icons
  1. <script>
  2. $(window).load(function() {
  3.   $('div.slideshow').slideshow( { playClass: "font-icon-play", pauseClass: "font-icon-pause" });
  4. });
  5. </script>

Rhythnic Image Slideshow is Open Source

The bulk of the code for this plugin is from the Tuts Premium course, 30 Days to Learn jQuery. The course is free at the time I'm writing this, and it really taught me a lot about jQuery and about how to organize javascript code using dictionaries. I highly recommend it. I added the font icons to the tutorial code and possibly the autoplay, I can't remember.

This plugin is open source and there is a public repository for it on Bitbucket called image-slideshow. I will continue to develop it here and there in the future as time allows, but anyone is able to fork the git project and continue development. There are many image slideshow plugins around, but I chose to create one more for the purpose of understanding the process of making a plugin.

The Rhythnic Image Slideshow jQuery Plugin is release under the open source software license. Anyone may download it for free, change it at will, and use it for commercial purposes.
