About Rhythnic Image Slider

The Rhythnic Image Slider is a javascript component for converting image lists into slideshows. It doesn't rely on jQuery, so it will use less bandwidth than if it did. Users of touch screens can navigate the slideshow by swiping left or right. Desktop users will see the navigation icons appear when their mouse hovers over the image. This slider depends on Modernizr's detection of touch events to customize the look. Setting up the Rhythnic Image Slider does require a little knowledge of HTML and CSS.

How to Use the Image Slider Plugin

Putting the Rhythnic Image Slider on your page is a combination of HTML, CSS, and a little bit of javascript. Below I've broken it down into those sections so that you know exactly what to do to put the slider on your page.

The Zip File

The zip file contains the following files and folders:

Index.html

The HTML

Let's go over the code for the HTML file that will include the slider. We'll start at the top and work down.

The HEAD
Any build of Modernizr that tests for touch will do. You can also swap out the icon font system by specifying the play and pause class names in the options object when creating the slider in js.

The HEAD: CSS, Modernizr, Font Awesome
  1. <head>
  2.   <meta charset="UTF-8">
  3.   <meta name="viewport" content="width=device-width,initial-scale=1" />
  4.   <title>Your page title</title>
  5.   <link href="//netdna.bootstrapcdn.com/font-awesome/4.0.3/css/font-awesome.css" rel="stylesheet">
  6.   <link href="/css/rhythnic-slider.css" rel="stylesheet">
  7.   <link href="/css/main.css" rel="stylesheet">
  8.   <script src="/js/modernizr.custom.78546.js"></script>
  9. </head>

Let's look at our list of images next. The slider works by manipulating a list on your site. Include the list inside of a div with a class of rhythnic-slider and place that div on your page wherever you want the slider to be. Rhytnic's slider is made for use with images, but you could tweak it for other purposes. Any number of images is fine, but they should be the same size. Set the tabindex to 0 so that the slider can be selected via the tab button. If the slider has focus, you can use the space bar, left and right arrow keys to play and navigate the slideshow.

A list of images inside of a div with a class of rhythnic-slider
  1. <div class="rhythnic-slider" tabindex="0">
  2.   <ul>
  3.     <li><img src="/img/image1.jpg" alt="Description of image one"></li>
  4.     <li><img src="/img/image2.jpg" alt="Description of image two"></li>
  5.     <li><img src="/img/image3.jpg" alt="Description of image three"></li>
  6.     <li><img src="/img/image4.jpg" alt="Description of image four"></li>
  7.   </ul>
  8. </div>

If you want captions, just include a caption inside of a paragraph tag inside of the list element.

An image with a caption
  1. <li>
  2.   <img src="/img/image1.jpg" alt="Description of image one">
  3.   <p>Caption for image one.</p>
  4. </li>

Copy this div for the slider navigation and paste it below the unordered list of images, but still inside of the div with a class of rhythnic-slider.

Slider navigation div
  1. <div class="slider-nav">
  2.   <i class="skip fa fa-chevron-left" data-dir="prev"></i>
  3.   <i class="play fa fa-play"></i>
  4.   <i class="skip fa fa-chevron-right" data-dir="next"></i>
  5. </div>

The Rhythnic Image Slider uses Font Awesome, a popular brand of font icons, which work by assigning classes to i elements. Any i element in the slider with an "fa" class is a Font Awesome icon. The other class names and the "data-dir" attribute assist in the functioning of the plugin. I chose Font Awesome because it's free and very user friendly.

I chose to include the entire Font Awesome collection by linking to it in the head section for a few reasons, even though we're downloading more than is necessary. The first reason is that it keeps the slider package simple. Also, the less I integrate the font icon system, the easier it is for developers to integrate the slider plugin with the other font icons on their site, whether they are using Font Awesome or another icon font system. You're welcome to use another icon font system, but you have to change the class names to match whatever you decide to use. You'll also need to specify the new class names in the options object when you create the slider object. This is explained better in the options section below. Beginning developers will also benifit from the simplicity of linking to Font Awesome in the head.

The CSS

The rule for the slider container is at the top of rhythnic-slider.css. Use it to manipulate the size and the positioning of the slider. Don't change position relative and overflow scroll.

.rhythnic-slider selector in your main css file
  1. .rhythnic-slider {
  2.   width: 600px; /* width of container */
  3.   font-size: 100%;  /* adjust to control the sizing of the icons and text in slider*/
  4.   position: relative;
  5.   overflow: scroll; /*For no-js browsers */
  6. }

Each image will automatically fit the width of the container, even as you change the size of the container (assuming js is enabled). For the best presentation, please make all of the images files the same width and height. In the absence of javascript, the slider is a horizontal scrolling list of images. In this instance, the images don't take the width of the slider container. They are just their natural size. If you'd like to alter their size in the event that there is no javascript, please add a css rule.

Force image size
  1. .rhythnic-slider img {
  2.   width: 600px; /* ideal display width of each image */
  3. }

There is also a CSS transition rule for controling the the image transition speed. Right now it's one second, but you can change it in the CSS.

The Javascript

Put rhythnic-slider.min.js, modernizr.custom.80160.js in your script directory (js) and link to them in the HTML. We already linked to Modernizr in the head. The best place to include scripts, Modernizr excluded, is at the bottom of the body container.

Link to rhythnic-slider.min.js
  1. <script src="/js/rhythnic-slider.min.js"></script>

This code will create the slider for all divs that have a class of rhythnic-slider The code is placed inside of window.onload to ensure that all the images have downloaded before the slider is created.

Create and initiate many sliders
  1. <script>
  2. window.onload = function() {
  3.   var sliderContainers = document.getElementsByClassName('rhythnic-slider');
  4.   var sliders = [];
  5.   for (var i = 0; i < sliderContainers.length; i++){
  6.     var slider = new Object(RhythnicImageSlider);
  7.     slider.init(sliderContainers[i]);
  8.     sliders.push(slider);
  9.   }
  10. };
  11. </script>

Options

Currently, there are 2 options to control playback. You can have the slider auto-start when the page loads and you can specify the time between each picture when the slider is on play. In regard to the options object, you can specify any options in any order, including or omitting any at will, as long as you use the correct key names, such as play and timer.

The values specified in this code are the default values. You can replace false with true to enable autoplay, and you can replace 5000 with however many milliseconds you would like to be in between each picture.

Create one slider and pass options
  1. <script>
  2. window.onload = function() {
  3.   var sliderContainer = document.getElementsByClassName('rhythnic-slider')[0];
  4.   var slider = new Object(RhythnicImageSlider);
  5.   slider.init(sliderContainer, { play: false, timer: 5000 });
  6. };
  7. </script>

If you have decided to use another font icon system, please specify the new play and pause class names in the options object so that the play and pause images can be toggled.

Specifying class names for play and pause icons
  1. <script>
  2. window.onload = function() {
  3.   var sliderContainer = document.getElementsByClassName('rhythnic-slider')[0];
  4.   var slider = new Object(RhythnicImageSlider);
  5.   slider.init(sliderContainer, { playClass: fa-play, pauseClass: fa-pause });
  6. };
  7. </script>

Rhythnic Image Slider is Open Source

The origins of this humble little slider are from the Tuts Premium course, 30 Days to Learn jQuery. The course is free at the time I'm writing this, and it really taught me a lot about jQuery and object literal notation for organizing javascript code. I highly recommend it.

This javascript component is open source and there is a public repository for it on Bitbucket called rhythnic/image-slider. I will continue to develop it here and there in the future as time allows, but anyone is able to fork the git project and continue development. There are many image slider plugins around, but I chose to create one more for the purpose of understanding how to do it. I also like having a slider plugin that doesn't use jQuery, in case I don't need jQuery elsewhere on a site.

The Rhythnic Image Slider is released under the MIT License for open source software. Anyone may download it for free, change it at will, and use it for commercial purposes.
